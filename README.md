JuLisp
======

Julia言語によるLisp実装の試み。

https://github.com/saka1029/JuLisp からのフォークです。
https://qiita.com/saka1029/items/e06a6c43b81a0efc4cc1 を参照ください。


## インストール

~~~
$ julia
pkg> add https://github.com/akimichi/JuLisp
~~~

## テスト

> julia
pkg> activate .
pkg> test
